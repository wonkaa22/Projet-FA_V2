Notiffi est un plugin qui propose une version modernisée du système des notifications.

    Visuellement plus complet, il permet d'afficher les dates des notifications ainsi que l'avatar du membre qui envoie la notification.

    Contrairement à la Toolbar par défaut, il donne accès à la structure HTML des notifications, ce qui permet de les placer n'importe où et sous n'importe quelle forme.

    Il ajoute quelques fonctionnalités supplémentaires, comme la possibilité de marquer toutes les notifications comme lues sans les supprimer, ou bien de tout supprimer d'un seul clic pour un ménage plus rapide.



Notiffi fonctionne sur toutes les versions de Forumactif, à l'exception de AwesomeBB (en cours).

Installer Notiffi

La toolbar doit être activée pour le bon fonctionnement du plugin. Elle est activée par défaut à la création de votre forum mais si vous rencontrez un problème, assurez-vous qu'elle le soit toujours en vous rendant dans votre panneau d'administration : Modules > Toolbar > Activer la toolbar : oui
Installer le plugin

Dans le template overall_footer_end copiez le code suivant juste avant la fermeture de la balise </body>

<!-- NOTIFFIN PLUGIN -->
 <script src="https://cdn.jsdelivr.net/gh/Kim-Bnx/Notiffi@main/notiffi/notiffi.min.js"></script>
 <!-- Initialisation de Notiffi -->
 <script>Notiffi.init();</script>

Le script de Notiffi vous est proposé en hébergement extérieur afin d'optimiser son fonctionnement et son installation. Ca fait moins de lignes de code à manipuler pour vous, et ça me permet de le mettre à jour plus facilement. Vous pouvez retrouver le code d'origine sur Github. Sentez-vous libre de l'héberger vous-même au besoin !
Intégrer le HTML

Positionnement

Il est recommandé d'installer la structure HTML dans des templates qui construisent l'ensemble des pages de votre forum pour avoir accès aux notifications sur toutes les pages.

Exemples :

    overall_header : le haut du forum
    overall_footer_end/begin le bas du forum

Par défaut, la structure HTML est placée en position: fixed en CSS. Ce qui veut dire que le HTML s'affranchit de son conteneur et qu'il prend pour repère la fenêtre de votre navigateur. C'est pour ça que l'emplacement HTML importe peu dans la hiérarchie de la page de votre forum.

Il est tout à fait possible d'intégrer le HTML dans un conteneur bien défini (ex: la barre de navigation) afin de l'intégrer à d'autres éléments de la structure de votre forum. N'oubliez pas de modifier le CSS en conséquence.
Modifier la structure

Vous remarquerez l'utilisation de variables entre crochets comme [count] ou [notifs]...[/notifs] qui fonctionnent comme les {variables} de Forumactif. Il s'agit de Potion, un moteur de template créé par @Poumon ✨

Pour assurer le bon fonctionnement de Notiffi, vous devez suivre quelques obligations :

    Le code HTML doit se trouver entre les commentaires <!-- BEGIN switch_user_logged_in --> et <!-- END switch_user_logged_in --> pour ne s'afficher que pour les membres connectés
    Ne retirez et ne modifiez pas les attributs type="" et data-name="" des balises &lt;template> : ils assurent l'affichage des données des notifications
    Ne retirez et ne modifiez pas les attributs id="" de certain éléments : ils assurent le bon fonctionnement de certaines interactions

Pour le reste, sentez-vous libre de restructurer le HTML selon vos envies ! De nouveaux éléments HTML peuvent être ajoutés et les existants peuvent être déplacés.

Options du plugin

Dans le template overall_footer_end vous avez installé ceci :

<!-- Initialisation de Notiffi -->
<script>Notiffi.init()</script>

C'est ici que les options doivent être ajoutées en respectant la syntaxe suivante :

<!-- Initialisation de Notiffi -->
<script>Notiffi.init({
 button: '...',
 panel: '...',
 timeout: 5000,
 icons: {
   0: '...'
 },
})</script>

Fonctionnement

Les options sont facultatives. Vides, Notiffi fonctionnera avec ses valeurs par défaut. Il est autant possible de changer qu'une option que toute celles proposés.

Les options qui attendent des types de valeurs très précises. Ici, vous allez en rencontrer quatre :

    string est une valeur en chaîne de caractère qui s’écrit entre doubles guillemets "". On peut y mettre du texte mais aussi du HTML. Attention ! Remplacez les guillemets de votre HTML par de simples apostrophes sinon votre string va se fermer trop tôt.
        Exemple : <i class="icon"></i> → <i class='icon'></i>
    number est une valeur composé uniquement de chiffre.
    boolean est une valeur binaire qui peut être soit true (vrai) soit false (faux), pour activer ou désactiver une option.
    object est un objet, une valeur entre crochets { } qui stock plusieurs valeurs associées en clé/valeur. Par exemple : { nom: 'Kim' }
        nom est la clé
        'Kim' est la valeur

Chaque valeur d'un objet est séparé par une virgule.
Options disponibles
button

Valeur par défaut : "#notiffi_button"

<script>Notiffi.init({
 button: '#notiffi_button',
})</script>

C'est le sélecteur de l'élément qui permet d'ouvrir le panneau des notifications au clique.
panel

Valeur par défaut : "#notiffi_panel"

<script>Notiffi.init({
 panel: '#notiffi_panel',
})</script>

C'est le sélecteur de l'élément qui contient les notifications et qui apparait et disparait au clique du bouton.
timeout

Valeur par défaut : 5000

<script>Notiffi.init({
 timeout: 5000,
})</script>

C'est le délais durant lequel l'alerte d'une nouvelle notification reste active. La valeur est en milliseconde, donc 1 seconde = 1000. Par défaut, l'alerte reste 5 secondes (5000) avant de disparaitre.
disableIcon

Valeur par défaut: false

<script>Notiffi.init({
 disableIcon: true,
})</script>

La mention true permet de désactiver l'affichage des icons, ce qui vous permet également de retirer de la variable [icon] dans votre HTML.
icons

Valeur par défaut : { ... }

<script>Notiffi.init({
 icons: {
   0: 'Message privé',
   1: '🚩',
   2: '<i class="bi bi-person-fill-add"></i>'
 }
})</script>

Cette option permet de changer le contenu de la variable [icon] qui change selon le type de la notification (un nouveau message privé, un report d'abus, une invitation, un tag...). Chaque type de notification est identifié par un ID (un chiffre). La liste ci-dessous indique quel type est associé à quel chiffre. Vous pouvez modifier le contenu par du texte, des emojis ou de l'HTML.

Par défaut, le plugin utilise des icons de la bibliothèque d'icone Bootstrap (installé dans le Blank Theme).
